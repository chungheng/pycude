PyCUDE¶

PyCUDE is yet another package of parallel implementation for the Differential Evolution algorithm. The parallelization assumes that the objective function can be evaluated in parallel with parameters presented in PyCUDA arrays.

This package is largely based on scipy’s implementation of DE algorithm. The algorithm is invented by Storn and Price.

Installation¶

Clone and navigate into the repository, and then execute the command:

python setup.py

This package depends on scipy and numpy.

Example¶

from pycude import differential_evolution

# a function with input arguments in pycuda.gpuarray
def func(x, y):
    pass

bounds = [(0,2), (0, 2)]
result = differential_evolution(func, bounds)
